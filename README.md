
# 📬 Smart Gmail Assistant
# 🚀 AI-Powered Email Prioritization & Calendar Automation

# 🌟 Overview

Smart Gmail Assistant is an AI-driven browser extension designed to reduce email overload by automatically identifying high-priority emails and converting them into actionable calendar reminders.

It intelligently detects deadlines, tasks, and important conversations, ensuring users never miss critical updates buried in their inbox.

# 🎯 Key Highlights:

🧠 Built an AI-based prioritization system for real-time email analysis

🔗 Integrated Gmail API + Google Calendar API for seamless workflow automation

🔐 Implemented secure OAuth 2.0 authentication from scratch

⚡ Developed a browser extension with real-time UI updates inside Gmail

🎯 Focused on productivity + user-centric design

# 🚀 Features
# 🔍 Smart Email Intelligence

Detects deadlines, meetings, and tasks from email content

Identifies high-priority emails using custom logic

# ⏰ Deadline Highlighting

Visually marks urgent emails inside Gmail UI

Helps users instantly focus on what matters

# 📅 Calendar Automation

Converts emails into calendar events/reminders

Saves time from manual scheduling

# 🔔 Smart Notifications

Alerts users about upcoming deadlines

Prevents missed opportunities

# 🔐 Secure Authentication

Google OAuth 2.0 integration

Safe access to Gmail & Calendar

# 🧩 Problem Statement

Important emails often get lost in cluttered inboxes, leading to missed deadlines and reduced productivity.

# 💡 Solution

This project transforms Gmail into an intelligent task manager by:

Filtering noise

Highlighting urgency

Automating scheduling

# 🛠️ Tech Stack

Frontend : HTML, CSS, JavaScript
APIs : Gmail API, Google Calendar API
Authentication : OAuth 2.0
Platform : Chrome Extension
